# Get-UnityDNSServer

## SYNOPSIS
Information about DNS Servers.

## SYNTAX

```
Get-UnityDNSServer [-session <Object>] [-ID <String[]>] [<CommonParameters>]
```

## DESCRIPTION
Information about DNS Servers.
You need to have an active session with the array.

## EXAMPLES

### -------------------------- EXEMPLE 1 --------------------------
```
Get-UnityDNSServer
```

Retrieve informations about the DNS server.
