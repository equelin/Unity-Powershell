# Disable-UnityFastCache

## SYNOPSIS
Disable FAST Cache.

## SYNTAX

```
Disable-UnityFastCache [[-session] <Object>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Disable FAST Cache.
You need to have an active session with the array.

## EXAMPLES

### -------------------------- EXEMPLE 1 --------------------------
```
Disable-UnityFastCache
```

Disable FAST Cache.
