# Get-UnityLicense

## SYNOPSIS
Queries the EMC Unity array to retrieve informations about licenses.

## SYNTAX

### Name (Default)
```
Get-UnityLicense [-session <Object>] [-Name <String[]>] [<CommonParameters>]
```

### ID
```
Get-UnityLicense [-session <Object>] [-ID <String[]>] [<CommonParameters>]
```

## DESCRIPTION
Querries the EMC Unity array to retrieve informations about licenses.
You need to have an active session with the array.

## EXAMPLES

### -------------------------- EXEMPLE 1 --------------------------
```
Get-UnityLicense
```

Retrieve information about all licenses
